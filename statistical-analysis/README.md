# Statistical Analysis

This directory contains implementations of statistical methods and econometric models commonly used in economic research.

## Projects

### 1. Regression Analysis
- **Description**: Implementation of various regression techniques
- **Skills**: OLS, fixed effects, robust standard errors
- **Files**: `regression_models.py`, `panel_data_analysis.R`

### 2. Hypothesis Testing
- **Description**: Examples of statistical hypothesis testing
- **Skills**: t-tests, ANOVA, non-parametric tests
- **Files**: `hypothesis_testing.R`

### 3. Time Series Analysis
- **Description**: Analysis of time series data
- **Skills**: ARIMA, stationarity tests, forecasting
- **Files**: `time_series_models.py`

## Tools and Libraries Used

- **Python**: statsmodels, scipy, scikit-learn
- **R**: lmtest, plm, sandwich, forecast

## Key Competencies Demonstrated

- Understanding of statistical inference
- Econometric modeling
- Model diagnostics and validation
- Interpretation of results
- Handling endogeneity and causality issues
