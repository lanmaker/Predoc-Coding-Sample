# Research Projects

This directory contains code from research-oriented projects, including replication studies and original analyses.

## Projects

### 1. Paper Replication Study
- **Description**: Replication of findings from an economics paper
- **Skills**: Data collection, econometric analysis, reproducibility
- **Files**: `replication_study/`

### 2. Original Analysis
- **Description**: Original research analysis on economic topics
- **Skills**: Research design, data analysis, result interpretation
- **Files**: `original_analysis/`

## Research Skills Demonstrated

- Reading and understanding academic papers
- Replicating published results
- Implementing econometric methodologies
- Writing clear documentation
- Reproducible research practices
- Critical thinking about methodology and results

## Notes on Research Ethics

All data sources are properly cited. Any proprietary or sensitive data has been excluded from this repository.
